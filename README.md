Ender FitText
-------
It's [FitText.js](http://fittextjs.com/) -- for [Ender](http://ender.no.de/)!

Use it:

``` js
$('#myAwesomeTitle').fitText(options);
```

See the full API on the [FitText.js Github page](https://github.com/davatron5000/FitText.js)
